# Security Policy

## Supported Versions
We currently support the latest version of Aegis Drive for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
Please report any security vulnerabilities directly to the maintainers. Do not open public issues for security vulnerabilities.
