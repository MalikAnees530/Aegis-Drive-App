# Contributing to Aegis Drive

Thank you for your interest in contributing to Aegis Drive! As a student-led project, we value all feedback and contributions.

## How to Contribute
1. **Report Bugs**: Use the Issues tab to report any bugs or unexpected behavior.
2. **Suggest Features**: Have an idea for the AI model or navigation system? Open an issue!
3. **Submit Code**: 
   - Fork the repository.
   - Create a feature branch (`git checkout -b feature/your-feature`).
   - Commit your changes.
   - Push to the branch.
   - Open a Pull Request.

## Code Standards
- Follow Kotlin coding conventions.
- Ensure all new features are tested on a physical device if they involve CameraX or MediaPipe.
- Maintain the Navy Bento UI aesthetic.
